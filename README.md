<h1 align="center">Emerging 编程语言</h1>    
<h3>可以直接接触底层的一个高级语言</h3></h3>

## 事先说明
此编程语言并非低级语言（低级语言举例：Assembly 汇编）
这是类似于 C、Zig（一个新兴的编程语言）的高级语言

## 编译 Emerging 源代码
编译 emerging.cpp:

```batch
cl /EHsc emerging.cpp /Feemerging.exe
```

编译 lexer.cpp:

```batch
cl /EHsc lexer.cpp /Feemerging-lexer.exe
```

这些命令须在 x64 Native Tools Command Prompt for VS 2022 （Visual Studio 开发人员命令提示符）

## 下载安装程序（Setup）
在此 GitHub 仓库点击 Releases → 标签 v3.0 → 下载 Emerging-Setup-x64.exe
先下载，再按照步骤安装
